Trivia Knights

Posted by Sohail on 1-28-2021 - 5:29pm

Project description

This project was made in Software Engineering course. Myself and four other students were grouped together to simulate an agile work environment with 3 week sprints. Our objective was simply to create an app.

As a group we decided to create a web app using REACT and Django. The premise was a trivia application that people could play with their friends, kind of like trivia night at local bars. Initially we planned to do like a multiplayer hosted session sort of thing but React proved to be more challenging than we anticipated.

We were succesful in standing up the web app however. You can see the final project results here. The app allows a user to create a profile and will generate a list of trivia questions based on the user's initial parameters.

Featured Project -

MSPI 1-Byte Minecraft Processor

Posted by Sohail on 1-30-2021 - 1:58pm

Summary

This was a project for Computer Architecture. I designed my own 1 byte processor architecture and implemented it in Minecraft. The premise was that learning the cycles of a processor might be less abstract if a student could watch the cycles take place and interact with it.

The project was a success in that I was actually able to program a short script in redstone and the my professor (Dr.Apan Qasem) offered me the opportunity to develop the idea. You can see the result of that work here. I also did a breakdown of the elements, which you can see here. Ultimately, the limitations of the minecraft lighting engine made the game quite laggy, so introducing more complexity did not seem feasible at the time.

Since then, Minecraft has released a new version of Minecraft that doesn't use Java RTE, and is thus somewhat faster at render times. The MSPI map file is completely incompatable, however, and several of the redstone mechanisms no longer work after updates to how redstone works. I'm not sure I'll ever be able to pick this one up again but it was a fun an interesting project for sure.
